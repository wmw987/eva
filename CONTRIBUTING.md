# Contributing

Eva is currently an early architecture and prototype skeleton.

Useful feedback areas:

- event schema design;
- temporal validity model;
- geospatial query patterns;
- provenance and confidence modeling;
- SQLite, PostGIS, and SpatiaLite storage design;
- MCP or tool interface design;
- privacy and safety review.

Please do not submit real operational data, scraped corpora, private logs,
credentials, tokens, or non-synthetic examples in issues or pull requests.
